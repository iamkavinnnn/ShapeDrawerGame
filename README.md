# ShapeDrawerGame
This is a simple Shape drawing program that can be used to create shapes such as rectangles, circles, and lines. 

The program allows to:
 - create shapes through a click.
 - Select the shapes that has been created.
 - Delete the shapes.
 - Save the file to the computer.
 - Upload the drawing of the saved file from the computer.

Feel free to download this code and have a go.
